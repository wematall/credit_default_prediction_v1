# credit_default_prediction_v1
модель для банка
прогнозирование невыполнения обязательств по кредиту

Требуется, 
на основании имеющихся данных о клиентах банка, построить модель, 
используя обучающий датасет, для прогнозирования невыполнения долговых обязательств по текущему кредиту.
Выполнить прогноз для примеров из тестового датасета.
